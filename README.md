  <summary>Tic, Tac, Toe.</summary>
  
  #### ttt
  This is a [Tic-tac-toe](https://en.wikipedia.org/wiki/Tic-tac-toe) game with the usage of an [embed](https://discord.com/safety/using-webhooks-and-embeds#title-4) and [buttons](https://support-dev.discord.com/hc/en-us/articles/6381892888087-Bots-Buttons).
  
  `.ttt @user` / `.ttt`
  
  ##### Arguments
  - Member. The member you want to play with, defaults to bot if not provided.
